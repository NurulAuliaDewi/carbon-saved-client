# Dashboard Client

Aplikasi dashboard client untuk sistem carbon saved.

## 📋 Persyaratan Sistem

Pastikan sistem Anda memenuhi persyaratan berikut:

- **Node.js versi 18.XX.XX atau lebih tinggi**
- **Git** untuk clone repository
- **NPM** (sudah termasuk dengan Node.js)

### Cek Versi Node.js

Untuk memastikan versi Node.js yang terinstall:

```bash
node --version
```

## 🚀 Instalasi dan Setup

### 1. Clone Repository

Clone repository dari GitHub:

```bash
git clone https://github.com/NurulAuliaDewi/carbon-saved-client
```

### 2. Masuk ke Direktori Project

```bash
cd carbon-saved-client
```

### 3. Install Dependencies

Install semua package yang diperlukan:

```bash
npm install
```

### 4. Menjalankan Aplikasi Development

Compile dan jalankan aplikasi dengan hot-reload untuk development:

```bash
npm run serve
```

Aplikasi akan tersedia di: `http://localhost:8080`

## 📝 Perintah Tersedia

| Command | Deskripsi |
|---------|-----------|
| `npm install` | Install semua dependencies project |
| `npm run serve` | Jalankan development server dengan hot-reload |
| `npm run build` | Build aplikasi untuk production |
| `npm run lint` | Cek dan perbaiki kode sesuai style guide |

## 🔧 Development

Setelah menjalankan `npm run serve`, aplikasi akan:
- **Hot-reload**: Otomatis refresh saat ada perubahan kode
- **Live compilation**: Compile ulang secara real-time
- **Error reporting**: Menampilkan error di browser dan console
